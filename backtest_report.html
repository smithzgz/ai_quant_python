# Grafana Dashboard 配置

1. 数据源配置:
   - 名称: PostgreSQL
   - 类型: PostgreSQL
   - Host: localhost
   - Port: 5432
   - Database: quant_db
   - User: postgres
   - Password: postgres

2. 已创建的Dashboard:
   - data_overview.json (数据概览)
   - backtest_analysis.json (回测分析)

3. 快速启动命令:
   docker-compose up -d grafana
   # 访问 http://localhost:3000 (admin/admin)

4. 关键SQL查询:
   -- 回测运行列表
   SELECT * FROM backtest_runs ORDER BY created_at DESC;

   -- 净值曲线数据
   SELECT timestamp, equity_value FROM equity_curves WHERE run_id=?;

   -- 交易记录
   SELECT * FROM trade_records WHERE run_id=? ORDER BY entry_time;
